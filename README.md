# 🛒 E-Commerce Analytics Dashboard (2016–2018)

Dashboard interaktif untuk menganalisis performa penjualan e-commerce berdasarkan dataset Brazilian E-Commerce.  
Dibuat menggunakan Python dan Streamlit sebagai bagian dari submission **Belajar Fundamental Analisis Data (Dicoding)**.

---

## 📊 Fitur Dashboard

Dashboard ini menyediakan analisis utama berikut:

- 📈 Tren jumlah order dan revenue per bulan
- 🏙️ Analisis kota dengan transaksi & revenue tertinggi
- 📦 Performa kategori produk (volume & revenue)
- 💳 Analisis metode pembayaran
- 🎛️ Filter interaktif (tahun & Top N)

## 🚀 Cara Menjalankan Dashboard

1. Clone repository:
git clone https://github.com/salsabilahauliyazahrah ecommerce-analytics-dashboard-streamlit.git

2. Masuk ke folder project: 
cd ecommerce-analytics-dashboard-streamlit

3. Install dependencies:
pip install -r requirements.txt

4. Jalankan dashboard:
python -m streamlit run dashboard/dashboard.py

5. Buka di browser:
link: ecommerce-analytics-dashboard-auliya-cdc31.streamlit.app

## 💡 Insight Utama

- Tren penjualan meningkat signifikan sepanjang 2017 hingga awal 2018
- Kota besar seperti São Paulo mendominasi transaksi dan revenue
- Beberapa kategori unggul dalam volume, sementara lainnya unggul dalam revenue
- Metode pembayaran **credit card** paling dominan
